## Abstract

Modern software development increasingly relies on CI/CD pipelines such as GitHub Actions to automate build, test, and deployment processes. However, research indicates that over 60% of these pipelines contain critical misconfigurations, creating significant supply chain attack vectors. Organisations lack integrated tooling to detect workflow-level vulnerabilities, objectively measure their DevSecOps maturity, perform threat modelling within development workflows, predict pipeline outcomes, and quantify delivery performance — forcing reliance on fragmented, manual approaches that fail to scale.

This project presents **WithOps**, an intelligent DevSecOps platform that unifies five critical capabilities into a single, developer-friendly system: automated GitHub Actions workflow security analysis detecting 25+ vulnerability patterns, OWASP DSOMM-based maturity scoring across five security dimensions, AI-assisted threat modelling using STRIDE, CIA, and LINDDUN frameworks via an interactive visual canvas, machine learning-based CI/CD pipeline outcome prediction, and a DORA (DevOps Research and Assessment) metrics dashboard computing Deployment Frequency, Lead Time for Changes, Change Failure Rate, and Mean Time to Recovery from real-time GitHub webhook events.

The platform was implemented as a production-grade microservices architecture comprising nine independently deployable FastAPI services, a SvelteKit 5 frontend application, a Kong API gateway, and a full observability stack (Prometheus, Grafana, Jaeger, Loki), orchestrated via Docker Compose. The system integrates multiple AI providers — Anthropic Claude for vision-based threat analysis, Meta Llama 3 via Groq for fast inference, and Ollama for local embedding generation — alongside a Retrieval-Augmented Generation (RAG) system using Qdrant vector database for context-aware conversational AI. Real-time data collection is achieved through GitHub webhooks processed via a Redis pub/sub event bus, enabling fully automated DORA metric computation and pipeline prediction feedback loops.

A novel contribution of this work is the **DORA × DSOMM correlation analysis**, which quantitatively measures the relationship between an organisation's security maturity and its delivery performance within a single integrated platform. This capability addresses a gap in both academic literature and commercial tooling, where security assessment and delivery metrics have traditionally been treated as separate concerns. Early evidence from the platform suggests that security investment and delivery velocity are complementary rather than antagonistic.

All eight project objectives were achieved, and the platform is publicly deployed at https://app.withops.com/. The project demonstrates that a unified, AI-enhanced DevSecOps intelligence platform can substantially automate security assessment processes that previously demanded significant manual effort, while simultaneously providing quantitative delivery metrics that enable evidence-based process improvement.
